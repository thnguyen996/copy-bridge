CopyBridge

Peer-to-peer text transfer between devices

How to Use

1 Device A: Click Create Room to get a 4-character code
2 Device B: Enter the code and click Join Room
3 Paste text on either device and click Send
4 Recipient clicks Copy to Clipboard to copy received text
Privacy: All data transfers directly between devices. Nothing is stored on any server.
